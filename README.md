# Car Price Prediction using Machine Learning

## Author
Arya Paresh Pendbhaje

## Domain
Data Science / Machine Learning

## Project Overview
This project predicts the selling price of used cars using a Linear Regression model.
The model is trained on a real-world dataset and developed using Google Colab.

## Features Used
- Year of Manufacture
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner Type

- Categorical Encoding
